How does it work?

Qovery Engine is an open-source abstraction layer product that turns easy apps deployment on AWS, GCP, Azure and others Cloud providers. The Qovery Engine is coded in Rust and takes advantage of Terraform, Helm, Kubectl, Docker to manage resources.

Usage

Initialize EKS (AWS Kubernetes) and ECR (AWS container registry) on AWS


        
let engine = Engine::new(
    context, // parameters
    local_docker, // initialize Docker as a Build Platform
    ecr, // initialize Elastic Container Registry
    aws, // initialize AWS account
    cloudflare, // initialize Cloudflare as DNS Nameservers
);

let session = match engine.session() {
    Ok(session) => session, // get the session
    Err(config_error) => panic!("configuration error {:?}", config_error),
};

let mut tx = session.transaction();

// create EKS (AWS managed Kubernetes cluster)
tx.create_kubernetes(&eks);

// create the infrastructure and wait for the result
match tx.commit() { 
    TransactionResult::Ok => println!("OK"),
    TransactionResult::Rollback(commit_err) => println!("ERROR but rollback OK"), 
    TransactionResult::UnrecoverableError(commit_err, rollback_err) => println!("FATAL ERROR")
};

Resiliency is key

Qovery Engine knows when something goes wrong on the deployment of your applications. Qovery Engine is built with resiliency in mind.

A transactional engine inspired by what is provided into ACID databases has been developed at the heart of the product to rollback on a consistent and working application version when something goes wrong.

What is Qovery?

Qovery is an innovative solution that radically simplifies application deployment on the Cloud. Designed as an open-source abstraction layer, Qovery allows developers to easily deploy their applications on platforms like AWS, GCP, Azure, and more in just a few minutes. Written in Rust, the Qovery Engine leverages powerful tools such as Terraform, Helm, Kubectl, and Docker to efficiently orchestrate resources and Cloud infrastructures.

With Qovery, users benefit from automated, secure infrastructure management without worrying about the complex details of provisioning resources or configuring Kubernetes clusters. By natively supporting Kubernetes, Qovery combines the power of this orchestrator with an intuitive, simplified interface that is accessible via a powerful CLI or a user-friendly web interface. The platform also offers multi-cloud support, giving businesses the flexibility to operate across different Cloud providers based on their needs. Additionally, Qovery places a strong emphasis on security and encourages responsible disclosure to ensure user trust. Whether you are a startup or an established enterprise, Qovery is a turnkey solution that accelerates your deployments while optimizing costs and enhancing productivity.
